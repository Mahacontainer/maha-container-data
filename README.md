## เกี่ยวกับสินค้าและบริการของเรา

### สินค้าของเรา

- ตู้คอนเทนเนอร์มือ 1 (New Containers)
  ตู้ใหม่สภาพสมบูรณ์ แข็งแรง ได้มาตรฐานสากล มีหลายขนาดให้เลือก

- ตู้คอนเทนเนอร์มือสอง (Used Containers)
  มีทั้งเกรด Cargo Worthy (CW) และ One Way Trip คุณภาพดี พร้อมใช้งาน

- ตู้คอนเทนเนอร์ดัดแปลง (Modified Containers)
  รับออกแบบและดัดแปลงเป็นสำนักงาน ห้องเก็บของ ร้านค้า และคาเฟ่
  สามารถปรับแต่งตามความต้องการของลูกค้า

- ตู้เย็นคอนเทนเนอร์ (Refrigerated Containers / Reefer)
  จำหน่ายตู้ควบคุมอุณหภูมิ เหมาะสำหรับสินค้าแช่เย็นและแช่แข็ง

### บริการให้เช่า (Rental Services)

- เช่าตู้สำนักงาน (Office Container Rental)
  พร้อมระบบไฟฟ้าและเครื่องปรับอากาศ

- เช่าตู้ห้องน้ำ (Restroom Container Rental)
  เหมาะสำหรับไซต์งานก่อสร้าง งานอีเวนต์ และพื้นที่ชั่วคราว

- เช่าตู้ผ้าใบ (Tarpaulin Container Rental)
  สำหรับงานขนส่งและจัดเก็บสินค้า

- เช่าตู้เย็นคอนเทนเนอร์ (Refrigerated Container Rental)
  ควบคุมอุณหภูมิได้ตามความต้องการสินค้า
